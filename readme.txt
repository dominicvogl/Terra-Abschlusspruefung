README

Die Desktop Version der Webseite ist für Firefox 4.0.1 mit einer Auflösung von 1280 x 1024 Pixeln optimiert.

Die mobile Version ist für das iPhone 3G, 3GS (Mobile Safari) mit einer Auflösung von 320 x 480 Pixeln
optimiert.

Zum Betrachten der Website auf einem PC/MAC einfach die "index.html" über "Datei öffnen" starten.

Die mobile Version kann z.B. mit der Adobe Device Central CS5 betrachtet und getestet werden.

Alternativ "http://terra.catalyst-interactive.com"

in die Browseradressleiste eingeben um die Website zu öffnen. Über Media Queries wird dann die für das jeweilige Gerät optimierte Seite angezeigt.
